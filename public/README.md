# Public Assets Directory

## Resume Upload

To add your resume to the portfolio:

1. **Upload your resume as `resume.pdf` in the `public/assets/` directory**
2. The resume button in the hero section will automatically link to `/assets/resume.pdf`
3. Supported formats: PDF (recommended), DOC, DOCX

### Current Resume Link
- Hero button links to: `/assets/resume.pdf`
- Full path: `public/assets/resume.pdf`

## Other Assets

## File Naming Conventions

- Use lowercase with hyphens: `my-resume.pdf`
- Be descriptive: `satyam-singh-resume-2024.pdf`
- Keep file sizes reasonable (< 5MB for PDFs)
