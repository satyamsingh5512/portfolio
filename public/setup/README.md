# Setup Files

This directory contains downloadable files for the VS Code setup guide.

## Files Needed:

### 1. fira-code.zip

Download Fira Code font from: https://github.com/tonsky/FiraCode/releases

- Download the latest release ZIP
- Rename it to `fira-code.zip`
- Place it in this directory

### 2. vsc-extensions.txt

Create a file with your VS Code extensions list.

To export your extensions:

```bash
code --list-extensions > vsc-extensions.txt
```

Or create manually with your preferred extensions:

```
esbenp.prettier-vscode
dbaeumer.vscode-eslint
bradlc.vscode-tailwindcss
PKief.material-icon-theme
sdras.night-owl
# Add more extensions...
```

## Usage

These files are referenced in the `/setup` page for users to download and set up their VS Code environment.
