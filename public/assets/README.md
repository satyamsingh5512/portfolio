# Assets Directory

## Professional Image Setup

To use your professional image with the PixelatedCanvas component:

1. Save your professional image as `satyam-professional.jpg` in this directory
2. The image will automatically be used by the PixelatedCanvas component
3. Recommended image size: 400x500 pixels or similar aspect ratio
4. Supported formats: JPG, PNG, WebP

## Current Images

- `satyam-avatar.png` - Current avatar image (being used as fallback)
- `satyam-professional.jpg` - Your professional image (add this file)

## PixelatedCanvas Features

The component includes:
- Interactive mouse distortion effects
- Swirl animation on hover
- Pixelated rendering with customizable cell size
- Smooth animations and transitions
