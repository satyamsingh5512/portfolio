## Project Overview

Quizo is an advanced, real-time quiz and assessment platform suitable for educational and corporate environments. It places a primary focus on maintaining quiz integrity and handling concurrent exam takers efficiently.

## Key Features

- **Anti-Cheat Measures**: Effectively tracks user tab switching and enforces strict browser event control to block copying.
- **Real-Time Data**: Employs WebSockets to instantly broadcast live scoreboard updates and handle time-limited questions.
- **Admin Management**: Easily manage tests, observe ongoing sessions, and access highly detailed user assessment reports.
