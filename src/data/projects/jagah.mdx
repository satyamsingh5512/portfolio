## Overview

Jagah is an Ephemeral AWS S3 Explorer. A highly secure, pure client-side application to manage AWS S3 buckets with zero backend. Built with a premium, responsive "B2B SaaS" UI aesthetic.

If you found this project helpful please give it a star on GitHub! It was made with the intention to view all the files you have stored in an S3 bucket easily. Its only motive is for developer preview.

## Architecture

Built with React and TypeScript, it uses the AWS SDK v3 for all interactions. It features a complete serverless architecture, functioning entirely within the browser client.

## Key Features

- **100% Ephemeral Security**: AWS credentials are kept strictly in volatile browser memory. Closing the tab instantly destroys access.
- **Full Bucket Management**: Browse folders, upload, rename, and delete files directly from the UI.
- **Secure Downloads**: Generates short-lived, authenticated presigned URLs on the fly for secure file downloads.
- **Premium Aesthetic**: Features a bespoke, dual-mode UI with crisp micro-animations, refined layout systems, and a warm "skin-tone" light mode.

## Tech Stack

### Core Stack
- **Library**: React
- **Language**: TypeScript
- **Cloud SDK**: AWS SDK v3
- **Styling**: Vanilla CSS
