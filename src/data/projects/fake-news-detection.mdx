# Fake News Detection Using NLP & LSTM

## Overview

Developed an LSTM-based text classification pipeline with GloVe embeddings achieving 89% F1-score on benchmark dataset for automated fake news detection.

## Key Features

### Advanced Text Preprocessing
- Text cleaning and normalization
- Stop word removal and stemming
- Tokenization and sequence padding
- Special character handling

### GloVe Word Embeddings
- Pre-trained GloVe embeddings integration
- 300-dimensional word vectors
- Semantic relationship preservation
- Out-of-vocabulary word handling

### LSTM Architecture
- Bidirectional LSTM layers
- Attention mechanisms for focus
- Dropout for regularization
- Dense layers for classification

### Feature Engineering
- N-gram analysis
- TF-IDF vectorization
- Sentiment analysis features
- Linguistic pattern extraction

## Tech Stack

- **NLP:** NLTK, spaCy, TextBlob
- **Deep Learning:** TensorFlow, Keras
- **Embeddings:** GloVe, Word2Vec
- **Data Processing:** Pandas, NumPy
- **Evaluation:** Scikit-learn

## Technical Implementation

### Data Pipeline
- News article collection and labeling
- Text preprocessing and cleaning
- Feature extraction and encoding
- Train-validation-test splitting

### Model Architecture
- Embedding layer with GloVe weights
- Bidirectional LSTM for sequence processing
- Attention layer for important word focus
- Dense layers with dropout for classification

### Training Strategy
- Binary classification (Real vs Fake)
- Cross-entropy loss optimization
- Adam optimizer with learning rate scheduling
- Early stopping and model checkpointing

## Performance Metrics

- **F1-Score:** 89% on benchmark dataset
- **Precision:** High precision for fake news detection
- **Recall:** Balanced recall across classes
- **Accuracy:** Consistent performance on test data

## Dataset & Evaluation

- **Benchmark Dataset:** Standard fake news detection dataset
- **Cross-Validation:** K-fold validation for robustness
- **Bias Analysis:** Evaluation across different news sources
- **Generalization:** Testing on unseen news domains

## Impact & Applications

- **Media Verification:** Automated fact-checking support
- **Social Media:** Real-time fake news flagging
- **Educational Tool:** Teaching media literacy
- **Research Contribution:** NLP methodology advancement

## Future Enhancements
