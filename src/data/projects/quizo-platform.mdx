# Quizo: Advanced Quiz Platform

## Overview

Developed a full-featured quiz platform with anti-cheat mechanisms, live score tracking, and session management, successfully serving 150+ active users.

## Key Features

### Anti-Cheat Mechanisms
- Tab switching detection
- Copy-paste prevention
- Time-based question delivery
- Browser focus monitoring
- Suspicious activity logging
- Automated violation handling

### Live Score Tracking
- Real-time leaderboard updates
- Instant score calculations
- Performance analytics
- Progress visualization
- Comparative analysis

### Advanced Session Management
- Secure user authentication
- Session timeout handling
- Multi-device login prevention
- Activity monitoring
- Automatic session recovery

### Quiz Management System
- Dynamic question loading
- Multiple question types support
- Time-bound assessments
- Randomized question order
- Difficulty-based categorization

## Tech Stack

- **Frontend:** React.js with modern hooks
- **Backend:** Node.js with Express.js
- **Database:** MongoDB for flexible data storage
- **Deployment:** Vercel for seamless hosting
- **Real-time:** WebSocket for live updates
- **Authentication:** JWT-based secure authentication
