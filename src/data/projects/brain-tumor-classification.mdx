# Brain Tumor Classification Using CNNs

## Overview

Built a CNN-based MRI classifier achieving 92% validation accuracy using transfer learning (VGG16/ResNet) and data augmentation for automated brain tumor detection.

## Key Features

### Transfer Learning Implementation
- Utilized pre-trained VGG16 and ResNet models
- Fine-tuned for medical image classification
- Leveraged ImageNet weights for better feature extraction
- Custom classification layers for tumor detection

### Data Augmentation Pipeline
- Rotation, scaling, and flipping transformations
- Brightness and contrast adjustments
- Noise injection for robustness
- Balanced dataset generation

### Model Architecture
- Convolutional layers for feature extraction
- Batch normalization for stable training
- Dropout layers for regularization
- Dense layers for final classification

### Performance Optimization
- Learning rate scheduling
- Early stopping mechanisms
- Model checkpointing
- Cross-validation for robust evaluation

## Tech Stack

- **Deep Learning:** TensorFlow, Keras
- **Computer Vision:** OpenCV, PIL
- **Data Processing:** NumPy, Pandas
- **Visualization:** Matplotlib, Seaborn
- **Model Architecture:** VGG16, ResNet

## Technical Implementation

### Data Preprocessing
- MRI image normalization and resizing
- Label encoding for multi-class classification
- Train-validation-test split strategy
- Data augmentation pipeline

### Model Training
- Transfer learning with frozen base layers
- Progressive unfreezing for fine-tuning
- Custom loss functions and metrics
- Hyperparameter optimization

### Evaluation Metrics
- **Accuracy:** 92% validation accuracy
- **Precision:** High precision across all classes
- **Recall:** Balanced recall for tumor detection
- **F1-Score:** Comprehensive performance evaluation

## Results & Impact

- **92% Validation Accuracy:** Achieved high accuracy on unseen data
- **Robust Performance:** Consistent results across different datasets
- **Medical Application:** Potential for clinical decision support
- **Scalable Solution:** Architecture suitable for deployment
